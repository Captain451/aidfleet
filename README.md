# AidFleet: A Smart Emergency Response System

## Overview
AidFleet is a web-based emergency response platform designed to connect emergency requesters with nearby ambulance drivers through real-time location tracking and intelligent dispatch support.

## Features
- User registration and authentication
- Driver registration and verification
- Emergency request submission
- Real-time location tracking
- Driver availability management
- Dispatch selection
- Admin dashboard and analytics

## Technologies Used
- PHP
- MySQL
- JavaScript
- HTML/CSS
- Google Maps API
- Africastalking API

## Installation
1. Clone the repository
2. Create a .env file
3. Import database/aidfleet.sql
4. Run Composer install
5. Configure environment variables
